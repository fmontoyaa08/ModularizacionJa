Este proyecto contiene un programa en java que demuestra el uso de metodos, para dividir la logica de una aplicaion.
El programa incluye un menú interactivo con las opciones:
1. Calculadora Basica.
2. Validacion de contraseña.
3. Verificacion de numeros primos
4. Suma de numeros pares
5. Conversion de temperatura.
6. contador de vocales en un texto
7. Calculo de factorial
8. Juego de adivinanza
9. Paso de párametros por referencia
10. Tablas de multiplicar
